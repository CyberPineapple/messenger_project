## Технологии

* Backend:
	* Language - python
	* Framework - aiohttp
	* Proxy - nginx
	* WSGI - gunicorn
	* DB - postgres (aiopg)
	* Key-value - redis (aioredis)
* Frontend:
	* Language - JS
	* Libraly - react + redux
	* Animate - animate-css
* Transport:
	* websockets
* Serialization:
	* JSON
